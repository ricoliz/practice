就活と親の役割

2013年08月30日

昨今の就活事情について、大学生たちが置かれている過酷な環境を他人事のように捕らえておりましたが、早いものでふと気が付けば、私の長男も現在大学3回生、まさに就活に突入しつつあります。
私は1985年に大学を卒業して銀行に就職し、現在に至りますが、私が就職活動をしていたころとはまるで状況が変わっているようです。エントリーシート然り、ウェブテスト然りです。そもそも当時は「就活」という言葉さえありませんでした。

「過保護」は論外ですが、「無関心」ではいられません。子供の人生の中で、大きな転機の一つとなる就活をミスリードしてはいけません。

親がすべきことは何か?　積極的に関わることは何か?
それは「親子間のコミュニケーション」に尽きると思います。キーワードは「コミュニケーション」です。
子供(学生)は大人(企業人)と話すことに慣れていません。狭い人間関係の中でしか会話をしていません。親は社会人の先輩として、企業人として、子供からアドバイスを求められるように、普段からしっかりと接し、コミュニケーションをとる必要があります。親と話すことで本人なりに「気付き」もあるでしょうし、親が社会人として話すことで、子供の「コミュニケーション能力」が鍛えられるはずです。企業は学生に「コミュニケーション能力」を求めていると思います。
また、「仕事とは何か」というような仕事観を伝え、理解し腹に落としてもらいたいものです。仕事の「やりがい」「社会貢献」「楽しさ・喜び」等を親が自分の意見として、また体験談として語ってやることに意味があると思います。

次に親がしてはいけないことは何か?
親の考え・価値観の押し付け、他人(同級生)との比較、今とは違う親世代の就活における常識の押し付け等ではないでしょうか。

基本的には、子供を自立させ、行き過ぎたサポート(過保護的支援)・関与は行わないということだと考えます。

とは言え、12月からが本番だとは言うものの、まさに始まっている長男の就活が気になってしかたがありません。まずはサマー･インターンシップでお手並み拝見。大学のある関西と関東を行ったり来たり。頑張れ我が息子 !

就活と親の役割(大阪バージョン)

8月某日
長男 : 「帰ってきたで、あーしんど、5時間もかかったがな。」
父親 : 「お帰り。大学生は夏休みでええなぁ。いつまで家におるんや。」
長男 : 「まだ決めてへん。適当におるわ。」
母親 : 「あんた、就活どないなってんねん。なんかしてるんかいな。」
長男 : 「ぼちぼちやわ。結構やってんで。」
父親 : 「サマー何とかあるんとちゃうんか。テレビでやってたで。」
長男 : 「いくつか、合格したで。東京でもうけるで。」
父親 : 「どんな業種狙ってんねん。もう絞ったんかいな。」
長男 : 「研究中。銀行、損保を考えてるねん。」
父親 : 「何でや。仕事の内容知ってんのか。」
長男 : 「ざっくりゆうたら、お金集めて貸してんとちゃうのん。」
母親 : 「あんた、えらいええかげんやなぁ。そんなんでええんかいな。」
長男 : 「冬からは本番で、エントリーシートや面接がきつなるわ。」
母親 : 「おとうさんに何か手伝ってもらいや。立ってる者は親でも使えてゆうからな。」
父親 : 「あほか。座ってるがな。」
長男 : 「エントリーシートの添削頼むわ。結構重要やで。」
父親 : 「任しとけ。ちゃんと見たる。どんな内容や。」
長男 : 「学生時代に力を入れたことや、ゼミの内容、クラブなんかや。」
父親 : 「そんなん本人しか分からへんやんけ。自分でやらんかい。無理やで。」
長男 : 「そんな殺生な。頼むがな。肩もむで。スペシャルバージョンやで。」
次男 : 「お兄ちゃん、キャッチボールしょう。」

今日も平和な家族であった・・・
